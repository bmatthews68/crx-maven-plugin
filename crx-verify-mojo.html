crx:crx-verify

Full name:

com.btmatthews.maven.plugins:crx-maven-plugin:1.2.1:crx-verify

Description:

Implement the crx-verify goal for the plug-in. The crx-verify goal verifies the signature of a Chrome Browser Extension.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.1.0.
  • Binds by default to the lifecycle phase: package.

Required Parameters

Name Type Since Description
finalName String 1.1.0 The final name of the artifact.
Default value is: ${project.build.finalName}.
outputDirectory File 1.1.0 The build target directory.
Default value is: ${project.build.directory}.

Optional Parameters

Name Type Since Description
classifier String 1.1.0 An optional classifier for the artifact.
crxPath File 1.1.0 The location of the Chrome Extension which is used to override the default location calculated using the output directory final name and classifier.
